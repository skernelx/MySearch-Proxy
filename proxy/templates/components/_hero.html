统一入口 网页发现、正文抓取、社交舆情走同一套接线方式。
Exa 工作区 把 Exa 独立成可发 Token、可管 Key、可直接代理调用的新栏目。
统一运维 Key 池、Token、额度同步和团队共享入口放在同一个控制台里。
统一兼容 官方 API 和自定义 compatible 聚合服务都能接，不锁死单一后端。
